# Video Meeting Platform

A Zoom-style video conferencing application built using React, Node.js, and LiveKit.

## Features

- Real-time video conferencing
- Microphone on/off
- Camera on/off
- Screen sharing
- Live chat
- Raise hand feature
- Active speaker detection
- Responsive video grid

## Tech Stack

- React.js
- Node.js (Express)
- LiveKit (WebRTC)

## How to Run

### Backend
cd backend
node server.js

### Frontend
cd frontend
npm install
npm start
